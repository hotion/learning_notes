2. 堆还是栈?

```go

func F() {
	a := make([]int, 0, 20) //栈内存
	b := make([]int, 0, 20000) //容量过大,堆内存

	l := 20
	c := make([]int, 0, l) //变长容量,堆内存
}

go build -gcflags='-m' . 2>&1  //查看
```
