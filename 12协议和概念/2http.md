## HTTP

#### HTTP 1.0

- 无状态、无连接

#### HTTP 1.1

- 持久连接
- 请求管道化
- 增加缓存处理（新的字段如cache-control）
- 增加Host字段、支持断点传输等

#### HTTP 2.0

- 二进制分帧
- 多路复用（或连接共享）
- 头部压缩
- 服务器推送
